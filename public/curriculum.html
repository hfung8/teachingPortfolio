Curriculum Unit

The Curriculum Unit was on Functional Programming and we used the programming language Racket to help demonstrate some aspects of Functional Programming such as: Recursive methods as ooposed to for loops, Pure functions as in functions that have no side effects (will always produce a certain output for a certain input), Immutability (since there is no state in functional programming, all functions will return a predictable outcome and we briefly discussed Higher Order functions such as map and filter.

Most of the information on Functional Programming was outlined in the Asynchronous Slides, we used the Synchronous ones for the lesson. We coded some examples of Racket with ones that used Conditionals and one which highlighted Functions in Racket during the Synchronous lesson. The homework had three different levels, Explorer, Adventurer and Pioneer. It focused on defining variables, creating functions and using recursion in Racket.

Overall, I enjoyed learning and coding in Racket. The process was slow but once I got the hang of it, there was a joy in understaning how to code properly. I think that this was another language, which was completely different from other ones that I had learned in the past, but one that I could see the benefits for learning it's differing paradigms and patterns.
